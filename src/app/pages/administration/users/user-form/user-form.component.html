Informations utilisateur
Règles de création
  • Le nom d'utilisateur doit être unique dans le système
  • Le CIN et le téléphone doivent contenir exactement 8 chiffres
  • Le rôle assigné doit être strictement inférieur à votre niveau actuel
  • Le département est obligatoire sauf pour les rôles ADMIN et SUPER_ADMIN
  • Le mot de passe sera généré automatiquement après la création de l'utilisateur
